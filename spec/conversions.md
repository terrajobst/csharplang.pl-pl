<span data-ttu-id="a40a1-591">Stosowanie kompilacji konwersja grupy metod `E` do typu delegata `D` opisano poniżej.</span><span class="sxs-lookup"><span data-stu-id="a40a1-591">The compile-time application of a conversion from a method group `E` to a delegate type `D` is described in the following.</span></span> <span data-ttu-id="a40a1-592">Należy pamiętać, że istnienie niejawna konwersja z `E` do `D` nie gwarantuje, że aplikacja kompilacji konwersji powiedzie się bez błędów.</span><span class="sxs-lookup"><span data-stu-id="a40a1-592">Note that the existence of an implicit conversion from `E` to `D` does not guarantee that the compile-time application of the conversion will succeed without error.</span></span>

*  <span data-ttu-id="a40a1-593">Jedna metoda `M` jest zaznaczone, odpowiadający wywołaniu metody ([wywołań metody opisywanego](expressions.md#method-invocations)) w postaci `E(A)`, z następującymi zmianami:</span><span class="sxs-lookup"><span data-stu-id="a40a1-593">A single method `M` is selected corresponding to a method invocation ([Method invocations](expressions.md#method-invocations)) of the form `E(A)`, with the following modifications:</span></span>
    * <span data-ttu-id="a40a1-594">Lista argumentów `A` znajduje się lista wyrażeń, wszystkich sklasyfikowanych jako zmienna oraz typ i modyfikator (`ref` lub `out`) z odpowiadającym mu parametrem w *formal_parameter_list* z `D`.</span><span class="sxs-lookup"><span data-stu-id="a40a1-594">The argument list `A` is a list of expressions, each classified as a variable and with the type and modifier (`ref` or `out`) of the corresponding parameter in the *formal_parameter_list* of `D`.</span></span>
    * <span data-ttu-id="a40a1-595">Metody Release candidate uważane za są tylko metody, które mają zastosowanie w ich normalnym formie ([dotyczy funkcja składowa](expressions.md#applicable-function-member)), wyklucza te mające zastosowanie tylko w postaci rozwinięty.</span><span class="sxs-lookup"><span data-stu-id="a40a1-595">The candidate methods considered are only those methods that are applicable in their normal form ([Applicable function member](expressions.md#applicable-function-member)), not those applicable only in their expanded form.</span></span>
*  <span data-ttu-id="a40a1-596">Jeśli algorytm [wywołań metody opisywanego](expressions.md#method-invocations) spowoduje błąd, a następnie wystąpi błąd kompilacji.</span><span class="sxs-lookup"><span data-stu-id="a40a1-596">If the algorithm of [Method invocations](expressions.md#method-invocations) produces an error, then a compile-time error occurs.</span></span> <span data-ttu-id="a40a1-597">W przeciwnym razie algorytm tworzy pojedynczy najlepszą metodę `M` mających taką samą liczbę parametrów jak `D` i konwersja jest uznawany za istnieje.</span><span class="sxs-lookup"><span data-stu-id="a40a1-597">Otherwise the algorithm produces a single best method `M` having the same number of parameters as `D` and the conversion is considered to exist.</span></span>
*  <span data-ttu-id="a40a1-598">Wybranej metody `M` musi być zgodny ([delegować zgodności](delegates.md#delegate-compatibility)) z typem delegata `D`, lub w przeciwnym razie wystąpi błąd kompilacji.</span><span class="sxs-lookup"><span data-stu-id="a40a1-598">The selected method `M` must be compatible ([Delegate compatibility](delegates.md#delegate-compatibility)) with the delegate type `D`, or otherwise, a compile-time error occurs.</span></span>
*  <span data-ttu-id="a40a1-599">Jeżeli wybrane metody `M` jest metodą wystąpienia, wyrażenie wystąpienia skojarzone z `E` określa obiektu docelowego delegata.</span><span class="sxs-lookup"><span data-stu-id="a40a1-599">If the selected method `M` is an instance method, the instance expression associated with `E` determines the target object of the delegate.</span></span>
*  <span data-ttu-id="a40a1-600">Jeśli wybranej metody M jest metodą rozszerzenia, która jest oznaczona za pomocą dostęp do elementu członkowskiego, na wyrażeniu wystąpienie, to wyrażenie wystąpienia określa obiektu docelowego delegata.</span><span class="sxs-lookup"><span data-stu-id="a40a1-600">If the selected method M is an extension method which is denoted by means of a member access on an instance expression, that instance expression determines the target object of the delegate.</span></span>
*  <span data-ttu-id="a40a1-601">Wynik konwersji jest wartość typu `D`, czyli nowo utworzony delegat, który odwołuje się do wybranego obiektu metody i docelowej.</span><span class="sxs-lookup"><span data-stu-id="a40a1-601">The result of the conversion is a value of type `D`, namely a newly created delegate that refers to the selected method and target object.</span></span>

<span data-ttu-id="a40a1-604">W poniższym przykładzie pokazano metodę konwersji grup:</span><span class="sxs-lookup"><span data-stu-id="a40a1-604">The following example demonstrates method group conversions:</span></span>
```csharp
delegate string D1(object o);

delegate object D2(string s);

delegate object D3();

delegate string D4(object o, params object[] a);

delegate string D5(int i);

class Test
{
    static string F(object o) {...}

    static void G() {
        D1 d1 = F;            // Ok
        D2 d2 = F;            // Ok
        D3 d3 = F;            // Error -- not applicable
        D4 d4 = F;            // Error -- not applicable in normal form
        D5 d5 = F;            // Error -- applicable but not compatible

    }
}
```

<span data-ttu-id="a40a1-605">Przypisanie do `d1` niejawnie konwertuje grupy metod `F` na wartość typu `D1`.</span><span class="sxs-lookup"><span data-stu-id="a40a1-605">The assignment to `d1` implicitly converts the method group `F` to a value of type `D1`.</span></span>

<span data-ttu-id="a40a1-606">Przypisanie do `d2` pokazuje, jak można utworzyć obiekt delegowany do metody, która ma mniej pochodnego (kontrawariantne) typy parametrów i bardziej pochodne typ zwracany (kowariantne).</span><span class="sxs-lookup"><span data-stu-id="a40a1-606">The assignment to `d2` shows how it is possible to create a delegate to a method that has less derived (contravariant) parameter types and a more derived (covariant) return type.</span></span>

<span data-ttu-id="a40a1-607">Przypisanie do `d3` pokazuje jak nie istnieje konwersja, jeśli metoda nie ma zastosowania.</span><span class="sxs-lookup"><span data-stu-id="a40a1-607">The assignment to `d3` shows how no conversion exists if the method is not applicable.</span></span>

<span data-ttu-id="a40a1-608">Przypisanie do `d4` pokazuje, jak metoda musi być stosowane w postaci normalny.</span><span class="sxs-lookup"><span data-stu-id="a40a1-608">The assignment to `d4` shows how the method must be applicable in its normal form.</span></span>

<span data-ttu-id="a40a1-609">Przypisanie do `d5` pokazuje, jak typy parametrów i zwrotu delegata i metody mogą różnią się tylko dla typów odwołań.</span><span class="sxs-lookup"><span data-stu-id="a40a1-609">The assignment to `d5` shows how parameter and return types of the delegate and method are allowed to differ only for reference types.</span></span>

<span data-ttu-id="a40a1-610">Podobnie jak w przypadku wszystkich innych Konwersje jawne i niejawne, aby jawnie przeprowadzić konwersję grupy metoda może służyć operatora rzutowania.</span><span class="sxs-lookup"><span data-stu-id="a40a1-610">As with all other implicit and explicit conversions, the cast operator can be used to explicitly perform a method group conversion.</span></span> <span data-ttu-id="a40a1-611">W związku z tym w tym przykładzie</span><span class="sxs-lookup"><span data-stu-id="a40a1-611">Thus, the example</span></span>

<span data-ttu-id="a40a1-613">Grupy metod może mieć wpływ na rozpoznanie przeciążenia i uczestniczyć w wnioskowanie o typie.</span><span class="sxs-lookup"><span data-stu-id="a40a1-613">Method groups may influence overload resolution, and participate in type inference.</span></span> <span data-ttu-id="a40a1-614">Zobacz [funkcji elementów członkowskich](expressions.md#function-members) więcej szczegółowych informacji.</span><span class="sxs-lookup"><span data-stu-id="a40a1-614">See [Function members](expressions.md#function-members) for further details.</span></span>

<span data-ttu-id="a40a1-615">Obliczanie czasu wykonywania metody konwersji grupy rozpoczynające się w następujący sposób:</span><span class="sxs-lookup"><span data-stu-id="a40a1-615">The run-time evaluation of a method group conversion proceeds as follows:</span></span>

*  <span data-ttu-id="a40a1-616">Jeśli metoda wybrana w czasie kompilacji jest metodą wystąpienia lub jest metodą rozszerzenia, który jest dostępny jako metodę wystąpienia, obiektu docelowego delegata jest określana na podstawie wyrażenia wystąpienia skojarzone z `E`:</span><span class="sxs-lookup"><span data-stu-id="a40a1-616">If the method selected at compile-time is an instance method, or it is an extension method which is accessed as an instance method, the target object of the delegate is determined from the instance expression associated with `E`:</span></span>

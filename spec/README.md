<a name="c-language-specification"></a>Specyfikacja języka C#
===========================

__W wersji 6__

Jest to nieoficjalny wersję roboczą, opublikowane w tym miejscu dla wygody. Po przesłaniu propozycja specyfikacji języka C# 6.0 ECMA (który jest aktualnie Trwa finalizowanie języka C# w wersji 5.0 Standard), będzie on udostępniany w tym miejscu.
